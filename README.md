# Philosophical Counselling Website

A modern Next.js website for a philosophical counselling practice, built with a calm premium visual language, reusable UI primitives, and a scalable App Router structure.

## Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- React Hook Form
- Zod

## Current Scope

- Modern homepage with reusable section architecture
- Public marketing pages for services, resources, FAQ, testimonials, and counsellor profile
- Accessible interactive FAQ accordion
- Validated contact form UI flow ready for backend integration
- Shared theme tokens and component primitives

## Run Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality Checks

```bash
npm run lint
npm run build
```
